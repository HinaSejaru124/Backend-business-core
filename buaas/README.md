# BuaaS — Business Assistance as a Service

Plateforme SaaS d'assistance aux conseillers d'orientation professionnelle.

## Stack
- Backend : Spring Boot 4 / Java 21
- Frontend : Next.js (SSR/PWA)
- Mobile : React Native
- Infra : Docker, Kafka, PostgreSQL, Redis
